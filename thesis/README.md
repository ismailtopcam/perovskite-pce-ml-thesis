# Tez Dokümanları

Bu klasör, tez metniyle ilgili Word, PDF veya yardımcı dokümanların saklanması için ayrılmıştır.

Örnek dosya türleri:

```text
*.docx
*.pdf
```

## Not

Tez dosyaları GitHub'a eklenmeden önce kişisel bilgiler, jüri bilgileri, ORCID, savunma tarihi ve benzeri alanlar kontrol edilmelidir.

Kod deposunda tez dosyasının bulunması zorunlu değildir. Tezde kullanılan hesaplamalı çıktıların tekrar üretilebilirliği için `src/`, `requirements.txt`, `README.md` ve seçilmiş `outputs/` dosyaları yeterlidir.
